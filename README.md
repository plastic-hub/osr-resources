# osr-resources
OSR resources, library, books, ...

### Installation 

1. Install [GIT LFS](https://git-lfs.github.com/) 'GIT LFS' enables large files over 50MB.
2. clone repo 


```
git clone https://github.com/plastic-hub/osr-resources 
```

3. In case files are not expanded, run

```
git lfs pull
```
